# White Editor Docs & Playground

White Editor를 소개하고 바로 써볼 수 있는 문서·플레이그라운드 입니다.

## Pages

###  **Home** 
 랜딩 페이지. 에디터 미리보기와 3D 오브젝트로 White Editor를 소개합니다. 

### **Playground** 
White Editor를 직접 사용해볼 수 있는 리치 텍스트 에디터. 헤딩, 리스트, 인용, 볼드/이탤릭/밑줄, 코드 블록, 링크, @mention, 이미지 등 기능을 체험할 수 있습니다.

### **Markdown**
마크다운을 입력하면 White Viewer로 실시간 미리보기, Copy·Reset 버튼으로 내용 복사·초기화가 가능합니다. 

### **Guide** 
White Editor 가이드 문서 페이지입니다.
